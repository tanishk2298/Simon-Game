Help!

Object of the Game:

Correctly repeat a longer and longer sequence of signals.

Instructions:

  1. Tap the play button to ON, the Game level 1 will apear and the Skill Level switch to 1,2, 3 or 4.
  2. Game will give the first signal. Repeat the signal by pressing the same color button.
  3. Game will add one. Repeat these two signals by pressing the same color button, in order.
  4. Game will add another one.
  5. Continue playing as long as you can repeat each sequence of buttons correctly.
  6. If you fail to repeat a sequence exactly Game responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.
